Reads in the first 2 bytes of the partition and makes sure it contains a valid magic word (f1f1).

If not the result output parameter is set to false.

Parameters
[in]-cmd This is the command section of the hole AT command.
[out]-result The result of the operation. True if a valid magic word is found and false if not.
Returns
- The status of the operation, See status_code_e for more information.

Special handling of the SYSFLASH command is needed as it returns data differently from normal text based commands.

Parameters
[in]-cmd This is the command section of the hole AT command.
[in]-param This is the parameter part of the AT command. The caller can fill a buffer using snprintf for instance with all the correct AT parameters.
[out]-retBuff This buffer will hold the result returned by the ESP-AT module.
[in]-timeout A timeout value can be specified if it is expected to take a very long time for the reply to arrive. This is 2 seconds default which probably covers most situations.
Returns
- The status of the operation, See status_code_e for more information.

Reads the PKI item from the specified partition and returns it in the provided buffer.

Parameters

[out]-pki_item The PKI item associated data such as PKI type, index and length.
