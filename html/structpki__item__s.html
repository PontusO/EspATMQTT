Detailed Description

Definition of the the PKI item header that makes out the first 4 bytes of each PKI item.

This shows s how a PKI partition is stored in flash memory:

+--------------+--------------+---------------+-------------------+---------------------------+
| <- 1 byte -> | <- 1 byte -> | <- 2 bytes -> | <- content len -> | <- (4-len%4) % 4 bytes -> |
+--------------+--------------+---------------+-------------------+---------------------------+
| type | ID | content len | raw data | padding for 4 bytes align |
+--------------+--------------+---------------+-------------------+---------------------------+

type: item type
0x01: CA
0x02: certificate
0x03: key
ID: used to match cert and key.
content len: raw data length
